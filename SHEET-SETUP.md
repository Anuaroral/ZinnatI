# Каталог ZINNÁTI · подключение Google-таблицы

Сайт сам подтягивает товары из опубликованной Google-таблицы (CSV). Пока ссылка не вставлена,
показываются демо-товары из `index.html` (массив `_fallback`).

## 1. Создать таблицу
1. Зайти на https://sheets.google.com → создать пустую таблицу.
2. File → Import → Upload → выбрать `catalog-template.csv` (он лежит в этой папке) →
   «Replace current sheet». Шапка и 16 примеров появятся автоматически.

## 2. Колонки (первая строка — ровно эти названия)
| колонка      | что писать | пример |
|--------------|------------|--------|
| `id`         | уникальный номер | `1` |
| `name`       | название | `Свитер «Astana»` |
| `cat`        | категория: `tops` `bottoms` `dresses` `cardigans` `sets` | `tops` |
| `price`      | цена в тенге (число) | `38500` |
| `sale`       | `TRUE` если на распродаже (→ блок «Весна / Sale»), иначе `FALSE` (→ «Лето / Новинки») | `TRUE` |
| `salePercent`| скидка в % (итоговая цена считается сама) | `20` |
| `oldPrice`   | старая цена для зачёркивания вручную (напр. для комплектов) | `68000` |
| `sizes`      | размеры через запятую | `XS,S,M,L` или `One size` |
| `stock`      | количество по размерам | `XS:2,S:5,M:0,L:3` |
| `badge`      | плашка: `ХИТ` / `NEW` / пусто (для sale ставится `−%` сама) | `NEW` |
| `img1`       | главное фото (ссылка или `images/имя.jpg`) | … |
| `img2`,`img3`| доп. фото 1–2 шт (img2 показывается при наведении) | … |
| `desc`       | короткое описание | `оверсайз, кашемир` |

Правила:
- если размер в `stock` = `0` → он показывается зачёркнутым (нет в наличии);
- если все размеры `0` → товар помечается «Нет в наличии»;
- `sale=TRUE` + `salePercent=20` при `price=80000` → на сайте `80 000 ₸` зачёркнуто, `64 000 ₸` итог.

## 3. Опубликовать
1. File → Share → **Publish to web**.
2. Слева выбрать нужный лист, справа формат **Comma-separated values (.csv)**.
3. Publish → скопировать ссылку (она кончается на `output=csv`).

## 4. Вставить ссылку в сайт
В `index.html` найти строку:
```js
const SHEET_URL = '';
```
и вставить ссылку:
```js
const SHEET_URL = 'https://docs.google.com/.../pub?gid=0&single=true&output=csv';
```
Готово — теперь правки в таблице появляются на сайте после обновления страницы.
